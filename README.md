# AI周报生成器 - 浏览器扩展插件

这是一个完全独立运行的浏览器扩展，让你在浏览器中快速生成专业周报，无需任何后端服务器。

## 功能特点

- **双模式运行**：支持本地模板生成和AI API生成两种模式
- **完全本地运行**：无需网络连接，无需启动Python后端
- **智能AI生成**：内置AI算法，根据输入内容智能生成周报
- **多行业风格**：支持互联网、国企、外企、体制内、自由职业者等5种风格
- **快速生成**：支持右键菜单快速生成，选中文本即可生成周报
- **历史记录**：自动保存生成的周报，方便查看和管理
- **数据安全**：所有数据存储在浏览器本地，不会上传到任何服务器
- **AI API集成**：支持OpenRouter、Together AI、Hugging Face等免费AI服务

## 安装方法

### Chrome浏览器：
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `browser-extension` 文件夹
6. 扩展将出现在浏览器工具栏中

### Edge浏览器：
1. 打开Edge浏览器
2. 在地址栏输入：`edge://extensions/`
3. 开启左下角的"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择 `browser-extension` 文件夹

## 使用方法

### 基本使用：
1. 点击浏览器工具栏中的AI周报生成器图标
2. 在弹出的窗口中填写工作内容（必填）
3. 可选填写遇到的问题和下周计划
4. 选择适合的行业风格
5. 点击"生成周报"按钮
6. 查看、复制或下载生成的周报

### 快速生成：
1. 在网页中选中工作内容文本
2. 右键点击，选择"生成AI周报" → "快速生成周报"
3. 扩展将使用选中文本生成周报并复制到剪贴板

### 键盘快捷键：
- **Ctrl+Enter**：生成周报
- **Ctrl+D**：填充演示数据
- **Ctrl+Shift+C**：清空表单

## 技术特点

- **Manifest V3**：使用最新的Chrome扩展规范
- **纯前端实现**：所有AI逻辑在浏览器中运行
- **Chrome存储API**：使用本地存储保存历史记录
- **响应式设计**：适配不同尺寸的弹出窗口
- **无障碍访问**：支持键盘操作和屏幕阅读器
- **多语言支持**：支持中文和英文界面
- **AI API集成**：支持多种免费AI服务提供商

## AI API配置指南

### 支持的AI服务提供商

1. **OpenRouter**（推荐）
   - 免费额度：每天100次请求
   - 获取API密钥：访问 https://openrouter.ai
   - 支持多种开源模型：Llama、Mistral、Qwen等

2. **Together AI**
   - 免费额度：新用户有免费试用额度
   - 获取API密钥：访问 https://together.ai
   - 支持多种开源模型

3. **Hugging Face**
   - 免费额度：有限制的免费使用
   - 获取API密钥：访问 https://huggingface.co
   - 需要创建Access Token

### 配置步骤

1. 点击扩展弹出窗口中的"设置"按钮
2. 选择AI服务提供商
3. 输入API密钥
4. 点击"测试连接"验证配置
5. 点击"启用AI模式"开始使用

### 注意事项

- AI模式需要网络连接才能使用
- API密钥存储在浏览器本地，不会上传到任何服务器
- 免费服务可能有使用限制，请查看各服务商的条款
- 扩展完全在浏览器中运行，无需任何外部依赖
- 所有数据存储在浏览器本地，清除浏览器数据会删除历史记录和API密钥
- 如需图标文件，请将16x16、32x32、48x48、128x128像素的PNG图标放入icons目录
